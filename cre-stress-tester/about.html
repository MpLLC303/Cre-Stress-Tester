About This Project

A student project that turned into a working tool.

§ 01 — The Origin

Where this came from.

I'm Micah Price, an undergraduate at the University of Colorado Boulder's Leeds School of Business, currently enrolled in BCOR 2205 — Introduction to Information Management and Analytics. The assignment that started this was to build a Custom GPT around any topic I chose. Most of my classmates built general-purpose chatbots. I wanted to build something an acquisitions analyst at a mid-market real estate private equity firm might actually open on their second monitor.

The result is this tool. It started as a simple Custom GPT configuration and grew into a full-stack web application as I wanted more control over the knowledge base, the output format, and the guardrails against hallucination. What you're using now is the production version — deployed on Vercel, backed by the Anthropic Claude API, and grounded in a curated six-file knowledge base built from public industry research.

§ 02 — What This Project Demonstrates

The capabilities this project evidences.

This is both a portfolio piece and a practice artifact. The capabilities it demonstrates:

  • Applied prompt engineering — a deliberate ten-technique stack, visible in the production system prompt and versioned through three measurable iterations.
  • Curated knowledge base construction — six reference files assembled from public CBRE, NAREIT, NCREIF, Green Street, and NAIOP research, with explicit verification gates before upload.
  • Full-stack deployment — HTML/CSS/JS frontend, Vercel serverless backend, Anthropic Claude API integration, rate limiting, error handling.
  • Cost-aware model selection — benchmarked Sonnet 4.6 against Haiku 4.5 on a four-persona test set; made an explicit tradeoff based on measured quality and latency data.
  • Test-driven iteration — four test personas with known answer keys, five-dimension scoring rubric, including a deliberate control case to measure over-flagging behavior.
§ 03 — What I'm Building Toward

Target roles.

I'm building toward internships and early-career roles in REIT acquisitions, private equity real estate, and corporate development. This tool is designed to speak directly to those audiences — it captures the kind of first-pass analytical thinking that distinguishes a junior analyst worth keeping from one who just runs numbers.
